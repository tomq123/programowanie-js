Program

  • Przegląd podstawowych struktur:
    deklaracje, wyrażenia, typy i widoczność danych, warunki, pętle, obsługa i generowanie wyjątków.
  • Wprowadzenie do DOM (Document Object Model):
    omówienie obiektów window, document, navigator. DOM XML. Zmiany w strukturze DOM dokumentu HTML. Obsługa i generowanie zdarzeń.
  • Obiektowość w JS – sposoby deklarowania funkcji, konstruktory, prototypy, format JSON
  • Debugowanie kodu (narzędzia dla deweloperów wbudowane w przeglądarkę) – monitorowanie zdarzeń, zmiany zawartości zmiennych, przerwania, breakpointy
  • Obsługa Ajax
  • Obsługa obiektów SVG i Canvas
